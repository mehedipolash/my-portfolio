# my-portfolio

Personal developer portfolio of **Md. Mehedi Hasan Polash** — built with React + Vite.

🔗 **Live projects, skills, and contact** are showcased on the site.

## Tech stack

- React 18
- Vite 6
- Vanilla CSS (custom black / red / yellow theme with light & dark mode)

## Getting started

```bash
npm install      # install dependencies
npm run dev      # start the dev server
npm run build    # production build
npm run preview  # preview the production build
```

## Project structure

```
src/
├── App.jsx      # all sections (Hero, About, Skills, Projects, Education, Contact)
├── data.js      # profile, skills, projects and education content
├── index.css    # theme + styles
└── main.jsx     # React entry point
asset/           # CV (PDF) and profile photo, served at the site root
```
